# Changelog

All notable changes to this project will be documented in this file.

## [0.0.5] - 2026-02-16

### Added
- Added `iconLabelSpacing` parameter to `FuturisticNavBar` to allow custom spacing between icons and labels.
- Integrated Item Spacing slider in the example app's customization panel.

## [0.0.2] - 2026-02-16

### Improved
- Comprehensive README with detailed descriptions for all 13 styles
- Full API reference documentation
- Custom icon and label usage examples
- Video demo

## [0.0.1] - 2026-02-16

### Added
- Initial release of Nav Bar
- 14 premium navigation styles (Liquid, Floating, Obsidian, Quantum, Prism, Hologram, Chronos, Synapse, Monolith, Singularity, Sumi, Cosmos, Kinetic, and Background Glow)
- Universal customization framework with `customColors` and `effectToggles`
- Custom icon and label widget support
- Physics-based animations with haptic feedback
- 7 icon animation types (Scale, Rotate, Slide, Fade, Flip, Bounce, Magnetic)
- 3 pre-built themes (Cyberpunk, Aurora, Molten)
- Notification badges support
- Fully responsive and optimized rendering
- Comprehensive example app demonstrating all features

### Features
- **Premium Visual Styles**: Choose from 14 distinct, hand-crafted navigation styles
- **Deep Customization**: Override any color or toggle any visual effect
- **Flexible Icons/Labels**: Use standard IconData/String or custom Widgets
- **Smooth Animations**: Physics-based elastic curves with haptic feedback
- **Production Ready**: Optimized performance with clamped opacity calculations
